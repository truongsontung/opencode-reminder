# BƯỚC 0 — XÁC NHẬN TOOL ĐÃ NẠP

Gọi:
```
reminder_list
```

Kết quả mong đợi:
- Trả về `No reminders.` (hoặc danh sách nếu còn mục cũ) — nghĩa là plugin đã nạp
  và 4 tool `reminder_add`, `reminder_list`, `reminder_done`, `reminder_del` sẵn sàng.

Nếu tool không tồn tại:
- Plugin chưa nạp. Cần mở lại phiên có nạp `opencode-reminders`.

Dừng. Không làm gì thêm.
